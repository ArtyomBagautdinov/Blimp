Доступные GET запросы:

/events - получить список всех событий

/events/id - получить событие с конкретным id

/blimpers - получить список всех пользователей

/blimpers/id - получить пользователя с конкретным id

/hobbies получить список всевозможных увлечений в базе данных

/blimperHobbies&blimperId=id - получить список увлечений пользователя с конкретным id

/eventHobbies&eventId=id получить список тем которым посвящено событие с конкретным id

Доступные DELETE запросы:

/deleteEvent - удалить событие с конкретным id из базы данных

Пример тела запроса:

{ "eventId": id }

/deleteBlimper - удалить пользователя с конкретным id из базы данных

Пример тела запроса:
